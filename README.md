# sublime-luatots  
> Quick translate Lua to TypeScripte in Sublime Text  

[![Sublime Text][sublime-image]](sublime-link)  

## How to use  
1. `Ctrl` + `Shift` + `P`, open command palette  
2. Type `Lua To Ts`
